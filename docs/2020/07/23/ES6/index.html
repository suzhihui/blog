变量声明

var

  • 污染全局变量
  • 没声明之前使用 会预先定义：变量提升
  • 可以被定义多次
  • 默认不会产生作用域

let

  • 不会污染全局变量
  • 不存在变量提升
  • 不能被重复定义
    • 同一作用域下不能重复定义
    • let + {} 可以产生一个作用域， 异步循环
